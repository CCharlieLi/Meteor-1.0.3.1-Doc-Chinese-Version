快速开始

以下内容适用于所有被支持的架构。

安装Meteor：

$  curl https://install.meteor.com | /bin/sh

创建一个项目：

$  meteor create myapp

本地运行：

$  cd myapp
$  meteor
#  Meteor server running on: http://localhost:3000/

在线发布服务（我们提供免费的在线发布服务）：

$  meteor deploy myapp.meteor.com
